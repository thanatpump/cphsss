# 🎴 คู่มือการตั้งค่า Card Reader

## ขั้นตอนการใช้งาน

### 1. รัน Card Reader Service

**วิธีที่ 1: ใช้ Batch File (แนะนำ)**
- ดับเบิลคลิก `start-read-card-service.bat`
- รอให้เห็นข้อความ "Server running on http://localhost:3002"

**วิธีที่ 2: ใช้ Command Prompt**
```bash
node read-card-service.js
```

### 2. รัน ThaiIDCardReader.exe (ถ้าต้องการอ่านบัตร)

**วิธีที่ 1: ใช้ Batch File (แนะนำ)**
- ดับเบิลคลิก `start-thaiid-reader.bat`

**วิธีที่ 2: รันโดยตรง**
- เปิดโฟลเดอร์ `card-reader-desktop\SSO-CardReader-For-Users\ThaiIDCardReader\`
- ดับเบิลคลิก `ThaiIDCardReader.exe`

### 3. เสียบเครื่องอ่านบัตร

1. เสียบเครื่องอ่านบัตรเข้า USB port
2. เสียบบัตรประชาชนเข้าเครื่องอ่านบัตร
3. เปิดหน้าเว็บ `allocation-check/data`
4. ระบบจะแสดงสถานะ "พบบัตรประชาชนในเครื่องอ่านบัตร"
5. กดปุ่ม "อ่านบัตรและเช็คสิทธิ์"

## ⚠️ ปัญหาที่พบบ่อย

### Error: ไม่สามารถเชื่อมต่อ Card Reader Service ได้

**สาเหตุ:** Card Reader Service ไม่ได้รันอยู่

**วิธีแก้:**
1. ตรวจสอบว่า Card Reader Service รันอยู่หรือไม่
   - ดูที่ Command Prompt ว่ามีข้อความ "Server running on http://localhost:3002" หรือไม่
2. ถ้าไม่รัน ให้รันใหม่:
   - ดับเบิลคลิก `start-read-card-service.bat`
   - หรือรัน `node read-card-service.js`
3. รอสักครู่แล้วรีเฟรชหน้าเว็บ

### Error: ไม่พบเครื่องอ่านบัตร

**สาเหตุ:** เครื่องอ่านบัตรไม่ได้เสียบ หรือ driver ไม่ได้ติดตั้ง

**วิธีแก้:**
1. ตรวจสอบว่าเครื่องอ่านบัตรเสียบเข้า USB แล้วหรือไม่
2. ตรวจสอบใน Device Manager ว่าเห็นเครื่องอ่านบัตรหรือไม่
   - กด `Win + X` → เลือก "Device Manager"
   - ดูที่ "Smart Card Readers"
3. ถ้าไม่เห็น ให้ติดตั้ง driver ของเครื่องอ่านบัตร

### Error: ไม่พบบัตรประชาชน

**สาเหตุ:** บัตรไม่ได้เสียบเข้าเครื่องอ่านบัตร

**วิธีแก้:**
1. ตรวจสอบว่าบัตรเสียบเข้าเครื่องอ่านบัตรแล้วหรือไม่
2. ลองดึงบัตรออกแล้วเสียบใหม่
3. ตรวจสอบว่าเครื่องอ่านบัตรทำงานปกติหรือไม่

## 📝 หมายเหตุ

- Card Reader Service ต้องรันอยู่ตลอดเวลาขณะใช้งาน
- ThaiIDCardReader.exe ต้องรันอยู่ถ้าต้องการอ่านบัตร
- ถ้าไม่ต้องการอ่านบัตร สามารถใช้วิธีกรอกเลขบัตรด้วยตนเองได้
