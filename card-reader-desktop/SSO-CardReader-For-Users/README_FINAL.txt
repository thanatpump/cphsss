═══════════════════════════════════════════════════════════
   SSO Card Reader - ไฟล์พร้อมใช้งาน
═══════════════════════════════════════════════════════════

📦 ไฟล์ในโฟลเดอร์นี้:

1. SSO-CardReader-Portable.exe - โปรแกรมหลัก (86 MB)
2. config.json - ไฟล์ตั้งค่า (URL: https://cphsss.hsoft.in.th)
3. คู่มือใช้งาน.txt - คู่มือใช้งานภาษาไทย
4. ThaiIDCardReader/ - โฟลเดอร์โปรแกรมอ่านบัตร

═══════════════════════════════════════════════════════════

🚀 วิธีใช้งาน (ง่ายมาก):

1. เปิดโปรแกรม ThaiIDCardReader.exe ก่อน
   → ดับเบิ้ลคลิกที่ไฟล์ ThaiIDCardReader.exe ในโฟลเดอร์ ThaiIDCardReader/

2. เปิดโปรแกรม SSO Card Reader
   → ดับเบิ้ลคลิกที่ไฟล์ SSO-CardReader-Portable.exe
   → จะมีหน้าต่างโปรแกรมขึ้นมา พร้อม LED Indicator

3. เสียบบัตรประชาชน
   → เสียบบัตรประชาชนเข้าเครื่องอ่านบัตร
   → LED จะเปลี่ยนเป็นสีน้ำเงิน (กำลังอ่าน)
   → เมื่ออ่านสำเร็จ LED จะเป็นสีเขียว และเปิดเว็บอัตโนมัติ

4. ดูผลการเช็คสิทธิ์
   → เว็บจะเปิดอัตโนมัติและแสดงข้อมูลสิทธิ์

═══════════════════════════════════════════════════════════

✅ สิ่งที่ต้องมี:

✓ เครื่องอ่านบัตรประชาชน (เสียบกับคอมพิวเตอร์)
✓ ThaiIDCardReader.exe กำลังรันอยู่ (https://localhost:8443)
✓ เชื่อมต่ออินเทอร์เน็ต

═══════════════════════════════════════════════════════════

💡 LED Indicator:

🟢 สีเขียว = พร้อมอ่านบัตร (เสียบบัตรได้เลย)
🟠 สีส้ม = กำลังตรวจสอบ
🔵 สีน้ำเงิน = กำลังอ่านบัตร
🔴 สีแดง = เกิดข้อผิดพลาด

═══════════════════════════════════════════════════════════

❓ ถ้ามีปัญหา:

1. ถ้าโปรแกรมไม่เปิด
   → ลองคลิกขวาที่ไฟล์ .exe แล้วเลือก "Run as Administrator"

2. ถ้าไม่พบเครื่องอ่านบัตร
   → ตรวจสอบว่า ThaiIDCardReader.exe เปิดอยู่หรือไม่
   → ตรวจสอบว่าเครื่องอ่านบัตรเสียบอยู่หรือไม่

3. ถ้าไม่สามารถเชื่อมต่อ Server ได้
   → ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
   → ตรวจสอบ URL ใน config.json ว่าถูกต้องหรือไม่

═══════════════════════════════════════════════════════════

📝 หมายเหตุ:

- โปรแกรมจะอ่านบัตรอัตโนมัติทุก 2 วินาที
- ไม่ต้องกดปุ่มอะไร เพียงแค่เสียบบัตร
- เมื่ออ่านบัตรสำเร็จ จะเปิดเว็บอัตโนมัติ
- หน้าเว็บจะอ่านบัตรและเช็คสิทธิ์อัตโนมัติ
