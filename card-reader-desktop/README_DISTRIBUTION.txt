========================================
  คู่มือการแจกจ่ายโปรแกรม
  SSO Card Reader Desktop App
========================================

📦 วิธีที่ 1: Copy โฟลเดอร์ไปเลย (แนะนำ)
========================================

1. Copy โฟลเดอร์ card-reader-desktop ทั้งหมดไปยังเครื่องที่จะใช้งาน
   รวมไฟล์:
   - main.js
   - package.json
   - config.json (ตั้งค่าแล้ว: https://cphsss.hsoft.in.th)
   - INSTALL_SIMPLE.bat
   - SETUP.bat
   - start.bat
   - TEST_CONNECTION.bat
   - โฟลเดอร์ ThaiIDCardReader/

2. บนเครื่องที่จะใช้งาน:
   a. ดับเบิ้ลคลิก INSTALL_SIMPLE.bat (ครั้งแรกเท่านั้น)
      - ติดตั้ง Node.js (ถ้ายังไม่มี)
      - ติดตั้ง dependencies อัตโนมัติ
   
   b. ดับเบิ้ลคลิก start.bat เพื่อรันโปรแกรม
      - โปรแกรมจะเชื่อมต่อไปที่ https://cphsss.hsoft.in.th อัตโนมัติ

3. รันโปรแกรม ThaiIDCardReader.exe ก่อน
   - โปรแกรมจะทำงานที่ https://localhost:8443

4. เสียบบัตรประชาชน
   - โปรแกรมจะอ่านและส่งข้อมูลไปยัง Server อัตโนมัติ

========================================

📦 วิธีที่ 2: สร้างไฟล์ .exe (รันได้เลย)
========================================

สำหรับผู้ใช้ที่ไม่ต้องการติดตั้ง Node.js:

1. บนเครื่องที่พัฒนา:
   - ดับเบิ้ลคลิก create-standalone.bat
   - รอให้สร้างไฟล์ SSO-CardReader.exe เสร็จ

2. Copy ไฟล์ต่อไปนี้ไปยังเครื่องที่จะใช้งาน:
   - SSO-CardReader.exe
   - โฟลเดอร์ ThaiIDCardReader/
   - config.json (ถ้าต้องการแก้ไข)

3. บนเครื่องที่จะใช้งาน:
   - ดับเบิ้ลคลิก SSO-CardReader.exe ได้เลย
   - ไม่ต้องติดตั้ง Node.js หรือ npm

========================================

📋 สรุปไฟล์ที่ต้อง Copy
========================================

วิธีที่ 1 (Copy โฟลเดอร์):
   ✅ card-reader-desktop/ ทั้งหมด

วิธีที่ 2 (ไฟล์ .exe):
   ✅ SSO-CardReader.exe
   ✅ ThaiIDCardReader/ (โฟลเดอร์)
   ✅ config.json (ถ้าต้องการแก้ไข)

⚠️ หมายเหตุ
========================================

- config.json ตั้งค่า Server URL แล้ว: https://cphsss.hsoft.in.th
- ถ้าต้องการเปลี่ยน Server URL ให้แก้ไข config.json
- ต้องมีโปรแกรม ThaiIDCardReader.exe รันอยู่ก่อน
- ต้องเชื่อมต่ออินเทอร์เน็ตเพื่อส่งข้อมูลไปยัง Server
