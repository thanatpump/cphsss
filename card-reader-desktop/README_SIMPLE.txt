========================================
  SSO Card Reader Desktop Application
  คู่มือการใช้งานแบบง่าย
========================================

📋 วิธีติดตั้งและใช้งาน (สำหรับผู้ใช้ทั่วไป):

1. ดับเบิ้ลคลิก "INSTALL_SIMPLE.bat" 
   - โปรแกรมจะติดตั้งทุกอย่างให้อัตโนมัติ
   - ถ้ายังไม่มี Node.js จะเปิดหน้าเว็บให้ดาวน์โหลด

2. หลังจากติดตั้งเสร็จ ดับเบิ้ลคลิก "start.bat" เพื่อรันโปรแกรม

3. ตรวจสอบว่าโปรแกรม ThaiIDCardReader.exe รันอยู่
   - โปรแกรมจะทำงานที่ https://localhost:8443

4. เสียบบัตรประชาชนเข้าเครื่องอ่านบัตร
   - โปรแกรมจะอ่านและส่งข้อมูลอัตโนมัติ

========================================

🔧 สำหรับผู้พัฒนา (สร้างไฟล์ .exe):

1. รัน "build-exe.bat" เพื่อสร้างไฟล์ SSO-CardReader.exe
2. ไฟล์ .exe จะรวมทุกอย่างไว้แล้ว ไม่ต้องติดตั้ง npm

========================================

⚠️ หมายเหตุ:
- โปรแกรมนี้ต้องใช้ Node.js (จะติดตั้งอัตโนมัติ)
- ต้องมีโปรแกรม ThaiIDCardReader.exe รันอยู่
- ต้องเชื่อมต่ออินเทอร์เน็ตเพื่อส่งข้อมูลไปยัง Server
