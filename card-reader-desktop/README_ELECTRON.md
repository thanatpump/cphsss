# 📦 SSO Card Reader - Electron GUI Version

## 🎯 สำหรับผู้ใช้ทั่วไป (ไม่ต้องติดตั้ง Node.js)

โปรแกรม Desktop GUI ที่ใช้งานง่ายสำหรับผู้ใช้งานทั่วไป - เปิดแล้วทำงานอัตโนมัติ

### ✨ คุณสมบัติ:

- ✅ **GUI ใช้งานง่าย** - แสดงสถานะชัดเจน ไม่ต้องใช้ Command Prompt
- ✅ **ทำงานอัตโนมัติ** - เปิดแล้วอ่านบัตรอัตโนมัติทุก 2 วินาที
- ✅ **แสดงสถานะ** - แสดงสถานะการเชื่อมต่อเครื่องอ่านบัตร, Server, และการอ่านบัตร
- ✅ **ไม่ต้องติดตั้ง Node.js** - ไฟล์ .exe ทำงานได้เลย
- ✅ **Portable Version** - ไม่ต้องติดตั้ง ดับเบิ้ลคลิกแล้วใช้งานได้เลย

---

## 📥 วิธีใช้งาน (สำหรับผู้ใช้):

### วิธีที่ 1: Portable Version (แนะนำ - ไม่ต้องติดตั้ง)

1. **ดาวน์โหลดไฟล์:**
   - `SSO-CardReader-Portable.exe` - ไฟล์หลัก
   - `config.json` - ไฟล์ตั้งค่า (ถ้ามี)

2. **วางไฟล์ในโฟลเดอร์เดียวกัน**

3. **แก้ไข `config.json`** (ถ้าจำเป็น):
   ```json
   {
     "sso_api_url": "https://cphsss.hsoft.in.th",
     "port": 3001,
     "thaiid_reader_url": "https://localhost:8443/smartcard/data/"
   }
   ```

4. **รัน ThaiIDCardReader.exe ก่อน** (ถ้ายังไม่มี)

5. **ดับเบิ้ลคลิก `SSO-CardReader-Portable.exe`** เพื่อเริ่มใช้งาน

6. **เสียบบัตรประชาชน** - โปรแกรมจะอ่านและส่งข้อมูลอัตโนมัติ

### วิธีที่ 2: Installer Version

1. **ดาวน์โหลดไฟล์:**
   - `SSO Card Reader Setup.exe` - Installer

2. **ดับเบิ้ลคลิก Installer** และทำตามขั้นตอนการติดตั้ง

3. **เปิดโปรแกรมจาก Desktop Shortcut**

4. **เสียบบัตรประชาชน** - โปรแกรมจะอ่านและส่งข้อมูลอัตโนมัติ

## 🔧 สำหรับ Developer - สร้างไฟล์ .exe ใหม่:

### ขั้นตอนที่ 1: ติดตั้ง Dependencies

```bash
cd card-reader-desktop
npm install
```

### ขั้นตอนที่ 2: สร้างไฟล์ .exe

**วิธีที่ 1: ใช้ Batch File (แนะนำ - ง่ายที่สุด)**
```bash
build-electron.bat
```

**วิธีที่ 2: ใช้ Command Line**
```bash
npm run build-win
```

### ขั้นตอนที่ 3: ไฟล์ที่สร้าง

ไฟล์จะอยู่ในโฟลเดอร์ `dist-electron/`:
- `SSO Card Reader Setup.exe` - Installer (ติดตั้งแล้วมี shortcut)
- `SSO-CardReader-Portable.exe` - Portable (ไม่ต้องติดตั้ง)

## 🔄 การทำงาน:

1. **เปิดโปรแกรม** → ตรวจสอบสถานะอัตโนมัติ
2. **เสียบบัตรประชาชน** → โปรแกรมอ่านอัตโนมัติทุก 2 วินาที
3. **ส่งข้อมูลไปยัง Server** → อัพเดทสถานะทันที
4. **แสดงข้อมูลบัตร** → แสดงข้อมูลที่อ่านได้

---

## ⚙️ การตั้งค่า:

แก้ไขไฟล์ `config.json`:

```json
{
  "sso_api_url": "https://cphsss.hsoft.in.th",
  "port": 3001,
  "thaiid_reader_url": "https://localhost:8443/smartcard/data/"
}
```

- `sso_api_url`: URL ของเว็บ SSO
- `port`: Port ของ Desktop App (ไม่ต้องเปลี่ยน)
- `thaiid_reader_url`: URL ของ ThaiIDCardReader (ไม่ต้องเปลี่ยน)

## 🐛 แก้ไขปัญหา:

### ปัญหา: ไม่พบเครื่องอ่านบัตร

**วิธีแก้:**
1. ตรวจสอบว่า ThaiIDCardReader.exe กำลังรันอยู่
2. ตรวจสอบว่าเครื่องอ่านบัตรเชื่อมต่ออยู่
3. เปิด Browser ไปที่ `https://localhost:8443/smartcard/data/` ดูว่าตอบกลับหรือไม่

### ปัญหา: ไม่สามารถเชื่อมต่อ Server ได้

**วิธีแก้:**
1. ตรวจสอบว่าเว็บ SSO ทำงานอยู่
2. ตรวจสอบ URL ใน `config.json` ว่าถูกต้องหรือไม่
3. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต

### ปัญหา: โปรแกรมไม่ทำงาน

**วิธีแก้:**
1. ตรวจสอบว่าไฟล์ `config.json` อยู่ในโฟลเดอร์เดียวกับ .exe
2. ตรวจสอบว่า ThaiIDCardReader.exe กำลังรันอยู่
3. ลองปิดและเปิดโปรแกรมใหม่
